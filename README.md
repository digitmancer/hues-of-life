# Shades of Life

Conway's Game of Life, now in color.

## Controls

<kbd>Space</kbd> to pause simulation.

Click and drag cursor to populate cells.
Set color of new cells via keyboard keys:

<kbd>R</kbd> - red  
<kbd>G</kbd> - green  
<kbd>B</kbd> - blue  
<kbd>C</kbd> - cyan  
<kbd>M</kbd> - magenta  
<kbd>Y</kbd> - yellow  
<kbd>K</kbd> - black  
<kbd>W</kbd> - white
